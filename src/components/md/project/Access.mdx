# Access - Scholarship Program

> Platform that showcases the Learning, Activities, and Events of the English Access Scholarship Program conducted by the US Embassy. It has features like event galleries, Activities and resource sharing.

![Access Home Page](/access/access.png)

## <Rocket className="inline-block w-7 h-7 mb-1 mr-2" /> Live Demo
**[Check out the live project here <ExternalLink className="inline-block w-5 h-5 mb-1 ml-1" />](https://access-g1.vercel.app/)**

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Google App Script

## Features
- Event galleries to showcase past events and activities.
- Resources for students to study.
- Responsive design for accessibility on all devices.
- Contact form for inquiries and support.

## Lets Understand
This is not a prety much complex website, It's just a simple website to showcase the activities and events of the Access Scholarship Program.

## Bad Part
This is not an open source project as it contains some sensitive data related to the US Embassy and the Access Scholarship Program.
