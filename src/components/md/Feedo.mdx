
# Feedo - Get Anonymous Feedbacks

> Feedo is plattform where you can get anonymous feedbacks from your friends, colleagues or customers. It is simple, easy to use and free.

![Feedo - Get Anonymous Feedbacks](/feedo/feedo.png)

## Technologies Used
- Next.js
- TypeScript
- Resend
- Tailwind CSS
- Shadcn UI
- React Hook Form
- Zod
- MongoDB
- Mongoose
- NextAuth.js

## Features
- Get anonymous feedbacks through just a link.
- Simple and user-friendly interface.
- Email verification while Registeration.
- Responsive design for all screen sizes.

## Lets Understand
This website has mainly two Portions
- Feedback Receiver
- Feedback Giver

### Feedback Receiver Portion
The feedback receiver portion allows users to create an account and receive anonymous feedbacks from others. \n
Just go to the [Feedo Website](https://feedo-anon.vercel.app/) and make an account.

![Feedo Registeration Page](/feedo/signup.png)

Once you've loggedin then you will be moved to your dashboard where you can get your unique feedback link to share with others. And also you can see all the feedbacks you've received so far.
![Feedo Dashboard Page](/feedo/dashboard.png)

### Feedback Giver Portion
The feedback giver portion allows users to give anonymous feedbacks to others. Not only by link user can be found by `Find User portion`

As shown in the below image, that you can also make it that you are not receiving feedbacks from anyone by disabling the `Allow Feedbacks` toggle.
![Feedo Allow Feedbacks Toggle](/feedo/find.png)
